## Setup

## Clone the repo

```
git clone https://github.com/NurettinYAKIT/dotfiles.git ~/.dotfiles
```

## Run the installation script

```
sh install.sh
```

## To update Brewfile

This command will create the Brewfile with descriptions

```
brew bundle dump --describe
```
